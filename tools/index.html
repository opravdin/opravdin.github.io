Tools Collection

A collection of useful web tools created by Opravdin

  • Salary Calculator

    Calculate your net salary after taxes in Russia, with support for comparing different salary scenarios.

    Open Tool

  • Party Expense Calculator

    Calculate who owes whom after a party by tracking individual expenses. Supports direct payments or payments through a budget keeper.

    Open Tool

  • Image Object Splitter

    Разделите изображение на отдельные объекты по цвету фона. Поддерживает пакетную обработку и скачивание результатов.

    Open Tool

  • Генератор точечных паттернов

    Создание паттернов с белыми точками на чёрном фоне для калибровки 3D сканеров. Поддерживает различные размеры точек и инверсию цветов.

  • Калькулятор компенсации комиссии

    Рассчитайте цену товара с учетом комиссии сайта объявлений при продаже с доставкой. Укажите желаемую сумму и получите готовый текст для объявления с ценой самовывоза.
